# Customer Shopping Behavior Analysis

## 📌 Project Overview

This project focuses on analyzing customer shopping behavior using **SQL** and **Power BI** to identify purchasing patterns, customer preferences, and sales trends. The analysis helps businesses understand customer behavior and make data-driven decisions to improve marketing strategies, product sales, and customer engagement.

---

## 🎯 Objectives

* Analyze customer purchasing behavior and shopping trends.
* Identify top-performing product categories and customer segments.
* Perform data cleaning and transformation using SQL.
* Build interactive dashboards in Power BI for visualization.
* Generate business insights for better decision-making.

---

## 🛠️ Tools & Technologies Used

* **SQL** – Data cleaning, transformation, and analysis
* **Power BI** – Data visualization and dashboard creation
* **CSV Dataset** – Source data for analysis

---

## 📂 Project Files

| File Name                                        | Description                                     |
| ------------------------------------------------ | ----------------------------------------------- |
| `customer_shopping_behavior.csv`                 | Raw dataset containing customer shopping data   |
| `customer_shopping_behavior_sql.sql`             | SQL queries used for data cleaning and analysis |
| `customer_shopping_behavior_project.pbix`        | Power BI dashboard project file                 |
| `Customer Shopping Behavior Analysis_Report.pdf` | Final project report with insights and findings |
